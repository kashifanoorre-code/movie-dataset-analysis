\f0\fs24 \cf0 # Movie Dataset Analysis\
\
## Project Overview\
This project analyzes a movie dataset using Python and Jupyter Notebook.\
\
## Tools Used\
- Python\
- Pandas\
- NumPy\
- Matplotlib\
- Seaborn\
\
## Questions Answered\
1. Most frequent genre\
2. Genre with highest votes\
3. Most popular movie and its genre\
4. Least popular movie and its genre\
5. Year with the most movies\
\
## Key Findings\
- Drama is the most frequent genre.\
- Drama received the highest popularity among genres.\
- Spider-Man: No Way Home was the most popular movie.\
- 2020 had the highest number of movies in the dataset.\
